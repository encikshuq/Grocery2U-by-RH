# Grocery2U by RH v1.2.2

Update dibuat:

1. Home
- Kad Belanja Mingguan dikemas kini supaya graf lebih jelas dan smart view.
- Graf menggunakan line/area chart SVG yang lebih kemas berbanding bar kecil.
- Ringkasan resit dan claim dipisahkan dalam box kecil.

2. Live Buying Progress
- Bila senarai dihantar untuk proses beli, Home memaparkan visual live 0–100%.
- Progress dikira berdasarkan jumlah barang yang ditanda `Dah Beli` berbanding jumlah barang dalam senarai.
- Ada progress bar dan circular percentage ring.

3. Setting
- Menu Family dipindahkan ke Setting.
- Master Senarai Barang untuk tambah/edit/on-off barang juga dipindahkan ke Setting.
- Bottom navigation kini lebih ringkas: Home, Senarai, Claim, Setting.

4. Family Limit
- Setiap akaun hanya boleh create maksimum 5 family.
- Jika sudah capai 5, app akan paparkan alert dan tidak tambah family baru.

5. Cache
- Version app dinaikkan kepada 1.2.2.
- Service worker cache name dikemas kini supaya telefon auto clear cache lama.
